# IDE Logos

Drop provider/editor logos in this directory for the workspace IDE launcher UI.

## File Naming

Use lowercase editor ids matching runtime registry ids:

- `vscode.svg`
- `cursor.svg`
- `windsurf.svg`
- `antigravity.svg`
- `zed.svg`
- `intellij.svg`
- `webstorm.svg`
- `pycharm.svg`
- `clion.svg`
- `goland.svg`
- `rustrover.svg`

PNG alternatives are also fine (`<id>.png`) if SVG is unavailable.

## Recommended Style

- Transparent background.
- Square artboard (64x64 or 128x128).
- Keep icon centered with small breathing room.
