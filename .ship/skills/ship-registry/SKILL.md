# Ship Registry

Use this skill when publishing, installing, or browsing Ship packages. Packages are Git repositories with a `.ship/ship.jsonc` manifest that declare skills and agents for others to consume.

See `references/docs/` for the full reference:

- **Registry** -- What the registry is and how packages work.
- **Publishing Packages** -- Exports, `ship publish`, versioning, content hashing.
- **Installing Packages** -- `ship add`, `ship install`, dependencies, lockfile.
