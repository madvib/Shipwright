## Summary

Generate a docs site from Ship project artifacts (vision, specs, ADRs, releases, and selected notes).

## Alpha goals

- Static site output from `.ship/project` and `.ship/workflow/specs`
- Configurable include/exclude rules
- Simple navigation by namespace and status

## Future ideas

- Versioned docs by release
- Search index generation
- Publish pipeline integration
