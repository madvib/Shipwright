# v0.2.0 — Revenue

**Goal:** Monetize the installed base. Team features, paid tiers, marketplace.

Milestone opens when v0.1.0 ships. Capabilities TBD based on what users actually do with v0.1.0.

## Rough Shape
- Billing — Stripe integration, usage-based or seat-based
- Team accounts — shared profiles, role-based access, audit log
- Profile marketplace — publish, discover, install profiles from the community
- Desktop revival — Tauri app, native UX for Studio
- Substrate features — capability map runtime, drift detection, session records as product
- Enterprise — SSO, private registries, on-prem MCP server
